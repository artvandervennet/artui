# Code of Conduct

This project follows the
[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good intent, and help others learn. Maintainers
will enforce the Covenant for everyone interacting in the project's spaces
(issues, pull requests, discussions, the docs site).

## Reporting

To report behavior that violates the Code of Conduct, contact the maintainer
listed in [CODEOWNERS](.github/CODEOWNERS) via the email on their GitHub
profile, or use GitHub's private reporting on the repository's Security tab.
Reports are handled confidentially.
